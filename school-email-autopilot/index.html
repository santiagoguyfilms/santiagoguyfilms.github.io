FAQ

Common Questions

What is Claude Cowork?
Claude Cowork is a desktop app from Anthropic that lets you run AI automations (called "skills") on your computer. Think of it as your AI assistant that can connect to Gmail, Google Calendar, Slack, and more. You install the School Email Autopilot skill inside Cowork, and it runs the automation for you.
Does this work with any school?
Yes. You enter your school's email domain during setup (like @yourschool.org) and the skill configures itself. It also scans for common school keywords, so it catches emails from platforms like ParentSquare, Remind, and ClassDojo that schools use.
How do I install it?
After purchase, you'll receive the .skill file via email. In Claude Cowork, go to Settings → Skills → drag the file in. Connect your Gmail, Calendar, and Slack connectors (free, takes 2 minutes each). Enter your school domain, and you're done.
Can I use it for multiple kids at different schools?
Absolutely. You can set up the skill with multiple school domains, or install separate instances for each school. Each one maintains its own calendar and digest, so nothing gets mixed up.
What if my school uses a different email system?
The skill scans your Gmail inbox — so as long as school emails arrive in your Gmail (or are forwarded there), it works. It doesn't matter what system the school uses to send the emails.
